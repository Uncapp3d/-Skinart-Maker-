# Skinart-Maker (Uploader) – Installation & Usage

## Installation:
Make sure you have Python installed: https://www.python.org/downloads/

Install the requirements by running install.bat.

## How to run:

Paste the skins (1–27) into the "skinart" folder.

Run start.bat.

Insert your Minecraft Bearer token: https://kqzz.github.io/mc-bearer-token/
 🔑

Choose between Slim and Classic model.

Wait a moment while the skins upload ⏳
